About

NOTER is an open source, minimalistic, themable markdown editor that seamlessly integrates with large-language models. If you happen to run ollama locally, NOTER will automatically connect to the instance. It is currently in alpha stage: you can download it and run it, but most functionality is still missing and the code is not thoroughly tested yet.

Installation

NOTER is available for MacOS, Windows and Linux - you can download it from our releases page.
