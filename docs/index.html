NOTER — write together

NOTER is a minimalistic, themable markdown editor that seamlessly integrates with large-language models, while respecting your privacy.

About

NOTER is an open source, minimalistic, themable markdown editor that seamlessly integrates with large-language models. If you happen to run ollama locally, NOTER will automatically connect to the instance. It is currently in alpha stage: you can download it and run it, but most functionality is still missing and the code is not thoroughly tested yet.

Installation

NOTER is available for MacOS, Windows and Linux - you can download it from our releases page.

Configuration

You can customize the editor in many ways, using a config.json file (in the current directory). Changes are applied at editor startup time.

{
  "window": {
    "opacity": 1,
    "width": 900,
    "height": 550
  },
  "font": {
    "colour": "blue",
    "size": "16px",
    "family": "monospace, monospace"
  },
  "background": {
    "colour": "white",
    "gradient": null,
    "image": null,
    "opacity": "100%"
  },
  "status-bar": {
    "font": {
      "colour": "black",
      "size": "14px",
      "family": "Arial"
    },
    "background": {
      "colour": "white"
    }
  },
  "preview": {
    "font": {
      "colour": "black",
      "size": "20px",
      "family": "Arial"
    },
    "background": {
      "colour": "blue"
    }
  }
}
